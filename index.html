Urdu-to-English Neural Machine Translation

A deep learning approach to cross-lingual translation using custom LSTM architecture

View on GitHub See It in Action

Project Overview

This project implements a neural machine translation system built from scratch using PyTorch. It features custom implementations of LSTM cells and attention mechanisms following the seq2seq paradigm, which is particularly effective for handling language pairs with different word orders and grammatical structures, such as Urdu and English.

Custom LSTM Implementation

Built from scratch with standard LSTM gates for efficient information flow and memory management.

Attention Mechanism

Bahdanau (additive) attention to focus on relevant parts of the source sentence during translation.

Beam Search Decoding

Maintains multiple translation hypotheses for more natural and accurate translations.

Model Architecture

Encoder-Decoder with Attention

The architecture consists of:

  • Encoder LSTM: Processes Urdu input text and creates contextualized representations of input tokens.
  • Attention Mechanism: Allows the decoder to focus on relevant parts of the source sentence.
  • Decoder LSTM: Generates English output tokens sequentially with attention-weighted context.
